--------------

1.  Move the bin using **Arrow Keys** or **A/D**.
    
2.  Trash items fall from the top.
    
3.  **Catch the correct item** to gain points.
    
4.  **Avoid incorrect items** (they reduce score).
    
5.  Game speed gradually increases to make it more challenging.
